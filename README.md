# YOLO Kidney Stone Detection

This project utilizes the YOLO (You Only Look Once) object detection algorithm to identify and localize kidney stones in medical images.

## Live Demo

You can try out the live web application here:
**[https://yolo-kidney-stone-1.onrender.com/](https://yolo-kidney-stone-1.onrender.com/)**

## Project Structure

- `web/` - Contains the source code for the frontend and backend web application.
- `kidney-stone-yolo12.ipynb` - Jupyter Notebook used for training and evaluating the YOLO model.
- `model_result/` - Contains the trained model weights and testing results.
- `runs/` - Directory containing YOLO training metrics and validation outputs.

## Running Locally

To run the web application on your local machine, please navigate to the `web/` directory for further instructions.
